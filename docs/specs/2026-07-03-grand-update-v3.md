## 1. 설계 원칙 (리서치 근거)

1. **Make It Bumpy** (Pecorella): 균질 곡선 금지. 마일스톤 점프(×2/×4)와 벽을 교차 배치.
2. **벽 리듬**: 첫 약한 벽 8~15분, 첫 프레스티지 적기 20~40분(+50~100% 보상), 강한 벽 60~120분에 선택지 2~3개 동반.
3. **보스 = DPS 체크**: `boss_HP = expected_DPS × 30s × factor` (일반 0.8~1.1 / 벽 1.2~1.8 / 관문 2.0~3.0). 실패는 페널티가 아니라 "목표 제시".
4. **언락 폭포**: 2분 퀘스트 → 3분 보스 게이트 → 7분 두 번째 성장축 → 30분 도감. "어제와 다른 목표".
5. **머지 6단계 피드백**: 드래그→예측→드롭→합성→보상→다음 행동. 합성 순간에 감각 3겹 이상(흡착+히트스톱+팝+파티클+사운드).
6. **레드닷 = actionable만**: "지금 누르면 이득"일 때만. 확인하면 소멸.
7. **아트디렉션 준수**: docs/ART-DIRECTION.md 팔레트·픽셀 규칙. 신규 UI 전부 나인슬라이스+비트맵 폰트.

## 2. Wave A — 전투 연출 대개편 "Orbiting Grimoires" (7/3 밤)

디렉터 확정 지시 반영. 체감 임팩트 최대 → 최우선.

### A1. 궤도 마법서 (핵심)
- 위저드 시전 모션 폐지 → **Idle 모션 상시** (Attack1 사용 중단).
- **장착된 책 6권이 위저드 주위를 타원 궤도로 공전** (원소색 글로우 + 톰 아이콘 스프라이트). 장착/해제가 전장에 즉시 반영 — "수량=화력"의 시각화 (ISM 검 궤도 문법).
- 시전: 궤도의 해당 책이 **플래시 → 원소 투사체가 책 위치에서 발사** → 잠깐 딤 후 복귀. 발사 원점이 지팡이가 아니라 궤도의 책.
- 구현: `BattleWizardView` cast 큐 재설계, 신규 `OrbitingTomesView` (슬롯 상태 구독), `BattleEffects.fireProjectile` 발사 원점 변경.

### A2. 투사체 타격 정합 (버그 근본 수정)
- 엔진 cast 이벤트에 `targetIndex` 포함 (engine/types.ts, battle.ts).
- 렌더는 발사 후 **매 프레임 타겟 실좌표로 유도(호밍 보간)** — 적이 움직여도 정확히 명중. 타겟 사망/부재 시 마지막 좌표로 소멸.
- 유닛테스트: cast 이벤트 스키마 계약 테스트.

### A3. 전장 밀도
- 웨이브 몹 수 5 → 스테이지 밴드별 6~9 (엔진 상수 + 시뮬 재검증).
- 몹 진형 다양화 (일렬 → 산개 3열), 히트 시 넉백 미세화.
- 처치 시 whitePop + 코인 아크는 유지, 폭사 파티클 소량 추가.

## 3. Wave B — 조작감·UX 개편 "Hands & Signals" (7/4)

### B1. 드래그 시스템 재작성
- Pointer 기반 유지 + **드래그 고스트** (커서 추종, finger offset -48px, 원본 딤 0.4).
- **스냅 프리뷰**: 가장 가까운 유효 셀 하이라이트. **머지 가능 대상은 골드 글로우 + 결과 Lv 프리뷰**.
- 무효 드롭: 원위치 복귀 애니(160ms Back.Out) + 셀 셰이크.
- 터치 타깃 ≥48px 확보.

### B2. 머지/소환/강화 juice
- 머지: 두 카드 중심 흡착(120ms) → 히트스톱(60ms) → 결과 scale 0.4→1.2→1.0 (Back.Out) → 원소색 파티클 12~20개 + 카메라 쉐이크 70ms/0.002 (10레벨 승급 머지는 120ms/0.004 + 사운드 피치 업).
- 소환: 카드 바운스 인(스케일 0→1.15→1) + 소환진 플래시.
- 슬롯 강화: 배수 뱃지 펀치 스케일 + 스파크.
- 골드 부족/미스매치: 기존 셰이크 유지 + 저피치 사운드.

### B3. 정보구조 재편
- 하단 탭 5개: **BOOKS / SKILLS / QUESTS(신규) / REBIRTH / RANKS**.
- **레드닷 시스템**: 강화 가능(골드 충분), 스킬 포인트 보유, 퀘스트 보상 수령 가능, 환생 이득 +50%↑. 탭과 버튼에 dot.
- 첫 화면 하단 공백 해소: 인벤토리 아래 **"다음 목표" CTA 스트립** (현재 퀘스트 1줄 + 보상).
- 신규 기능은 자물쇠+해금 조건 노출 → 해금 모달 → 첫 행동 유도 (Section 5 언락 스케줄).

### C1. 마일스톤 점프 (bumpy)
- 최고 책 레벨 10/20/30/40 달성 시 **전 화력 ×2** (톰 테마 승급과 동시 연출 — "승급 순간"이 사건이 됨).
- 위저드 레벨 10/20/30: 시전속도/치명 소폭 영구 보너스.

### C2. 보스 = 진짜 DPS 체크
- 보스 HP를 `기대DPS × 30s × factor`로 재산정: 일반 보스 1.0, **스테이지 5의 배수 "벽 보스" 1.5**, 10의 배수 "관문 보스" 2.2.
- 보스 실패 시: "필요 DPS" 명시 배너 + 추천 행동(머지/강화/특화) 표시 — 실패가 목표 제시가 되게.
- 보스 처치 보상 상향 (골드 ×15 유지 + 마일스톤 상자: 스킬 포인트/특화 재화).

### C3. 환생 리듬 조정
- 해금 스테이지 10 유지하되 **크리스탈 공식을 `floor((stage-8)^1.7/6)`로** — 첫 환생 적기가 st14~16(≈25~35분)에 오도록. 환생 패널에 "지금 환생 시 +X% (권장: +50%↑)" 게이지.
- 시뮬레이터에 wall_strength 메트릭(T_next/median_T_recent) 추가, 목표 곡선: 첫 약벽 8~12분, 첫 환생 25~35분, 강벽 60~90분. **전 상수 시뮬 검증 후 출고.**

## 5. Wave D — 콘텐츠 확장 + 캐릭터 특화 (7/5~7/6)

### D1. 퀘스트/업적 (목표 축 신설, P0)
- **메인 퀘스트 체인 ~20개**: "첫 머지 → Lv5 책 → 슬롯 6개 채우기 → 첫 벽보스 → 첫 특화 → 첫 환생 → …" 각각 골드/스킬P/특화재화 보상. 초반 혼란 해소 + 언락 폭포의 뼈대.
- **업적 ~15개** (누적 머지 수, 보스 킬, 도감 수집): 소액 재화 + RANKS 옆 과시.
- QUESTS 탭 + CTA 스트립 연동. 엔진에 quest 상태 추가 (세이브 스키마 v3 마이그레이션).

### D2. 톰 도감 (Grimoire Codex)
- 원소×티어(10레벨 단위 테마) 도달 기록 → **도감 1칸 = 영구 보너스** (예: 화염 T2 달성 = 화염 데미지 +3%).
- 중복 머지가 "낭비"가 아니라 도감 진행이 됨 (LoM handbook 문법).
- BOOKS 탭 내 서브뷰.

### D3. 캐릭터 특화 (디렉터 신규 요구) — 2단 구조 제안
- **① 원소 공명 (즉시, 장착 구성 = 빌드)**: 같은 원소 3권 장착 = 그 원소 공명 발동 (화염: 스플래시 대상 +1 / 냉기: 슬로우 강화·중첩 시 빙결 / 신성: 보스 데미지 +50%p). 6권 모노 = 공명 강화. 장착 선택이 곧 전술이 됨. 궤도 구체 색으로 빌드가 한눈에 보임.
- **② 아케인 특성 (성장 축)**: 위저드 Lv 8/16/24에서 특성 선택 (3지선다, 환생 시 무료 재선택): 예) "연쇄 시전(다음 시전 20% 가속)" / "황금 서고(골드 +15%)" / "원소 순환(공명 요구 3→2권)". 스킬 4종과 별개의 "선택" 재미.
- 스프라이트: 특화 원소에 따라 위저드 로브 팔레트 틴트 (풀 전직 스프라이트는 컷 — 에셋 리스크).

### D1'. 퀘스트/업적 볼륨 상향 (디렉터 볼륨 지시 반영)
- 메인 퀘스트 20 → **30개** (Day 1~7 구간까지: "환생 3회", "유물 5종", "st50" 등 장기 목표 포함).
- 업적 15 → **40개+** (누적 카운터형: 머지 500회, 보스 100킬, 환생 10회, 도감 완성도 — 자동 생성 가능한 티어형).

## 5.5 Wave E — 장기 축 신설 "Relics & Rifts" (7/6) ← 디렉터 볼륨 지시로 부활

### E1. 유물 (Arcane Relics) — 환생을 반복할 이유
- 환생 시 마나 크리스탈 일부로 **유물 소환** (10크리스탈/회): 12종 유물, 랜덤 획득, 중복 = 유물 레벨업 (효과 강화).
- 유물 예: "잿불 문장(화염 +8%/Lv)", "서리 렌즈(빙결 시간 +0.2s/Lv)", "황금 책갈피(골드 +6%/Lv)", "가속 모래시계(시전 -3%/Lv)", "심연의 눈(치명 피해 +10%p/Lv)" 등 — 공명/특성과 조합되는 빌드 재료.
- 유물 탭은 REBIRTH 탭 내 서브뷰 (환생→소환→장착 3슬롯).
- **효과**: 환생이 "초기화 비용"에서 "수집 기회"로 바뀜 — Day 2~7의 핵심 반복 루프 (ISM 아티팩트 문법).

### E2. 균열 던전 (Rifts) — 일일 재방문 루프
- 일일 입장 2회, 2종: **황금 균열**(3분 골드 러시 — 몹 웨이브 가속) / **시험의 균열**(보스 연전 DPS 체크 — 특성 재화 보상).
- 전투 씬 재사용(몹 구성·보상만 오버라이드) — 신규 씬 없이 엔진 모드 플래그로 구현.
- **효과**: "오늘 할 일"이 생김 (일일미션 없이 재방문 루프 확보).

## 5.6 Wave F — 동반자·생산·표현 축 (병렬 실행) ← 디렉터 지시: 컷 없음

> 2026-07-03 디렉터: "제외시키지 마라. 에이전트 기준으로 일정을 짜라." 일정 산정을 인간 공수에서 **병렬 위임 기준**으로 전환. 컷 기준은 시간이 아니라 품질 게이트(전 스위트 그린 + 검수 통과)다.

### F1. 패밀리어 (펫)
- 위저드를 따라다니는 소환수 1종+ (fairy/임프 계열 — 기존 dungeon-tileset 스프라이트 재활용, 라이선스 준수). 보조 DPS(위저드 화력의 %) + 골드 자동 줍기.
- 성장: 패밀리어 레벨(전투 EXP 공유) + 진화 2단계(도감/퀘스트 보상과 연계).
- 확장점: 감사 보고서의 "새 시스템 추가" 패턴 (engine/types→constants→actions→UI 패널).

### F2. 마나 광산 (Mine)
- 시간 기반 생산: 광산이 실시간으로 마나석 채굴(오프라인 포함, 서버 정산과 동일 원리). 마나석 = 유물 소환 보조 재화 + 특성 리롤 재화.
- 층 개념: 스테이지 마일스톤으로 광산 층 해금 → 채굴 속도 상승. "접속 간격" 루프 (LoM Mine 문법).

### F3. 위저드 스킨 + 일일미션
- 스킨 4종: 업적/도감/환생 마일스톤 보상 (로브 팔레트 스왑 — 특화 틴트와 별개 레이어). 과시+수집.
- 일일미션 5종 (로컬 자정 리셋, 운영 불필요): 머지 N회/보스 N킬/던전 입장/유물 소환/환생 — 보상은 마나석.

## 6. 일정 & 루프 계획

일정은 인간 공수가 아니라 **병렬 위임 + 통합 게이트** 기준이다. 구현 웨이브는 시간 단위, 병목은 게이트뿐.

| 배치 | 병렬 실행 (소유권 분할) | 통합 게이트 |
|---|---|---|
| 배치 1 (7/3 밤) | A 전투연출(game/scenes+engine이벤트) ∥ B UX(ui/) ∥ C 밸런스(engine/constants+simulate) | 머지 → 테스트·빌드·시뮬 곡선 → 배포 → TestSprite → 디렉터 대리 검수 |
| 배치 2 (7/4) | 엔진 스키마 v3 선행(1에이전트) → D 퀘스트·도감·특화 ∥ E 유물·던전 ∥ F 펫·광산·스킨·일일미션 (기능별 병렬) | 동일 게이트 + BE 세이브 v3 + FE 신규 플랜 |

- 각 웨이브: 구현(위임) → 디렉터 검수(스크린샷/플레이) → 배포 → TestSprite → LOOP.md 1줄.
